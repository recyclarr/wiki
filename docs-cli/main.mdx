## Usage

```js
recyclarr [OPTIONS] <COMMAND>
```

## Options

### `-v|--version` {#version}

Print version information for Recyclarr. This information is essential when reporting issues or
asking for help.

## Commands

- [recyclarr sync](/cli/sync)
- [recyclarr list](/cli/list)
- [recyclarr migrate](/cli/migrate)
- [recyclarr config](/cli/config)
