## Documentation Access

Use context7 MCP tools to retrieve current Docusaurus documentation. Always resolve the library ID
first, then fetch relevant docs.

Prefer results with `/website/` prefix for official documentation over blog posts or tutorials.

## Project Context

This is a Docusaurus v3 site with:

- MDX files in `docs/` directory (guide/, reference/, cli/)
- Custom components in `src/components/`
- SCSS styling in `src/css/`
- Multiple sidebars configured in `sidebars-*.js` files
- Partials using underscore prefix in `src/partials/`
