## Usage {/*#usage*/}

```js
recyclarr [OPTIONS] <COMMAND>
```

## Options {/*#options*/}

### `-v|--version` {/*#version*/}

Print version information for Recyclarr. This information is essential when reporting issues or
asking for help.
