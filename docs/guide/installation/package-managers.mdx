Installation via system package managers.

## Homebrew

[Homebrew][brew] is available for macOS and Linux.

```bash
brew tap recyclarr/recyclarr
brew install recyclarr
```

To upgrade:

```bash
brew update
brew upgrade recyclarr
```

Source: [github.com/recyclarr/homebrew-recyclarr][brew-tap]

[brew]: https://brew.sh/
[brew-tap]: https://github.com/recyclarr/homebrew-recyclarr
