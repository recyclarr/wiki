Each major upgrade of Recyclarr contains breaking changes. Please view the page for the version you
are upgrading to.

:::info

If you are upgrading between *multiple major versions*, you'll need to view the upgrade guides in
ascending order for each version up to the version you want to upgrade to.

For example, if you're currently on `v6.x` and plan to go to version `v8.x`, you'll need to view
the upgrade guides `v7.0` and `v8.0` in that order.

:::

## Retention Policy {#retention-policy}

Only the four latest major versions are kept here. Older upgrade guides are removed over time. If
you are upgrading from a version that is no longer listed, we recommend starting fresh by following
the [Getting Started][getting-started] guide with the latest version of Recyclarr.
