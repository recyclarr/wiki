Recyclarr automatically syncs recommended settings from [TRaSH Guides](https://trash-guides.info/)
to your Sonarr and Radarr instances. This guide walks you through installation to your first sync.

## Step 1: Install Recyclarr

Choose your preferred installation method:

<details>
<summary>Docker Compose (Recommended)</summary>

Create a `docker-compose.yml` file:

```yml
services:
  recyclarr:
    image: ghcr.io/recyclarr/recyclarr
    container_name: recyclarr
    user: 1000:1000
    volumes:
      - ./config:/config
    environment:
      TZ: America/Santiago
```

For networking setup, image tags, and advanced options, see the [Docker Compose guide][docker].

</details>

<details>
<summary>Homebrew (macOS & Linux)</summary>

```bash
brew tap recyclarr/recyclarr
brew install recyclarr
```

To upgrade later: `brew update && brew upgrade recyclarr`

Source: [github.com/recyclarr/homebrew-recyclarr][brew-tap]

</details>

<details>
<summary>Manual Download</summary>

Download the latest release for your platform:

<DownloadTable />

Extract and place the `recyclarr` binary somewhere in your `$PATH`.

For installation scripts and details, see the [Manual Installation guide][manual].

</details>

<details>
<summary>Unraid</summary>

Search for "recyclarr" in the Community Apps tab, or use the un.recyclarr plugin.

See the [Unraid guide][unraid] for details.

</details>

## Step 2: Create Configuration

<Tabs groupId="install-method">
  <TabItem value="native" label="Native" default>

```bash
recyclarr config create
```

  </TabItem>
  <TabItem value="docker" label="Docker">

```bash
docker compose run --rm recyclarr config create
```

  </TabItem>
</Tabs>

This creates `recyclarr.yml` in your [application data directory][appdata].

## Step 3: Edit Configuration

Open `recyclarr.yml` and configure at minimum:

- `base_url`: Your Sonarr/Radarr instance URL
- `api_key`: Found in Sonarr/Radarr under Settings â†’ General

See the [configuration reference][reference] and [examples][examples] for guidance.

## Step 4: Run Sync

<Tabs groupId="install-method">
  <TabItem value="native" label="Native" default>

```bash
recyclarr sync
```

  </TabItem>
  <TabItem value="docker" label="Docker">

To run manually:

```bash
docker compose run --rm recyclarr sync
```

Or start in [Cron Mode][cron] for automatic daily syncs:

```bash
docker compose up -d
```

  </TabItem>
</Tabs>
